Добрый день! Мой веб-сервис «Time For Family», который я разработал в рамках конкурса проектных работ для школы «Летово». Данный сервис поддерживает современную мобильную версию для любых портативных устройств.

Приходилось ли Вам сталкиваться с ситуацией, когда Вы не знаете, как провести выходной с семьей? Нет ничего проще! "Time For Family" поможет Вам решить эту проблему! С помощью языка Python 3 и веб-фреймворка Django мной был разработан веб-сервис для поиска наиболее удачных парков, аквапарков, кинотеатров, торговых центров и других мест, где можно увлекательно провести время!

SQLite база данных данного сервиса была сформирована с использованием Google Places API. Отправляя запросы с нужной тематикой и разбирая JSON-ответы от данного сервиса, мне удалось сформировать базу мест для заданной целевой аудитории - семейный отдых и отдых с детьми. Были собраны все данные о местах - названия, адрес, веб-сайт, рейтинг и даже отзывы! С помощью технологий HTML/CSS/JavaScript был сформирован дружелюбный для пользователя интерфейс, отображающий все эти данные для любых устройств. Следующей задачей стало использование машинного обучения и семантических библиотек для определения положительных и отрицательных отзывов, которые я получил от Google. Были использованы библиотеки TextBlob и Scikit-Learn. Далее, с помощью метода голосования, сервис может легко определить, каких отзывов больше - хороших или плохих. Эта возможность становится надежным помощником посетителя сайта при выборе места и становится основой для сервиса рекомендаций.
